## Installation
1. Clone the repository
    ```sh
    $ git clone git@github.com:kkznch/learn-react-without-create-react-app.git
    $ cd learn-react-without-create-react-app
    ```
2. Run npm install
    ```sh
    $ npm install
    ```

## Usage
```sh
$ npm run start
```
